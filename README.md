# 🧴 Body Lotion Dataset – Exploratory Data Analysis (EDA)

## 📌 Overview
This project explores **Nykaa’s Body Lotion dataset** to understand how **price, discounts, and ratings** influence customer choices.  
The analysis provides actionable insights for **pricing strategies, brand positioning, and marketing optimization**.

---

## 🎯 Problem Statement
Customers choose products based on **price, reviews, and brand**.  
We aim to analyze how these factors affect customer decisions and identify patterns that can guide business strategies.

---

## 🛠️ Tools & Libraries
- Python  
- Requests, BeautifulSoup (Web Scraping)  
- Pandas, NumPy (Data Cleaning & Manipulation)  
- Matplotlib, Seaborn (Visualization)  

---

## 📊 Dataset Summary
- **Records:** 784 products  
- **Features:** Product Name, Brand, Price, Selling Price, Discount, Savings, Price Category, Rating Count  
- **Price Range:** ₹100 – ₹1500  
- **Discounts:** 0% – 60%  

---

## 🔎 Methodology
1. **Web Scraping**: Extracted product details from Nykaa.  
2. **Data Cleaning**: Removed unwanted symbols (₹, %, “off”), handled missing values, removed duplicates.  
3. **Data Manipulation**: Added price categories (Budget, Mid-range, Premium), calculated savings, standardized brand names.  
4. **EDA**:  
   - Univariate Analysis → Histograms, Boxplots, Pie Charts  
   - Bivariate Analysis → Scatter Plots, Correlation, Boxplots  
   - Multivariate Analysis → Heatmaps, Correlation Matrix  

---

## 📈 Key Insights
- **Mid-range products** receive more ratings compared to budget products.  
- **Discounts** help budget brands compete but don’t guarantee high ratings.  
- **Strong brands** (e.g., Vaseline, Wishcare) win on trust rather than discounts.  
- Segmented pricing strategies allow brands to tap both budget and mid-range markets.
